# Pre Requisitos
- Realizar o git clone do projeto
- Acessar a pasta teste
- Executar o comando bundle install

# Executando os Testes
Executar o comando:
```
cucumber
```
Para execução com relatório:
```
cucumber -f pretty -f html -o report.html
```
